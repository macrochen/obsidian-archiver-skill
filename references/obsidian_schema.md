# Obsidian Archive Schema

## Metadata (Frontmatter)

All archived articles MUST include the following YAML frontmatter:

```yaml
---
title: "Article Title"
summary: "Brief summary of the article"
featureImage: "image_filename.png" # Optional
type: "WeChat" | "Xiaohongshu"
category: "Category Name"
date: YYYY-MM-DD
tags:
  - tag1
  - tag2
---
```

## File Naming Convention

Files should be named as: `Title.md`
- Example: `舒适达全系成分拆解.md`
- Example: `超市牙膏红黑榜.md`

## Content Structure

- Use Markdown headers (##, ###) for organization.
- Use dividers (***) between major sections.
- Use Callouts for tips or important notes if using Obsidian features.
- Preserve original formatting (tables, lists, bold text).
